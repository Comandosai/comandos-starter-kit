# COMANDOS Starter Kit — инструкция для резидента

> Один промпт — и у тебя разворачивается рабочий проект любого типа.
> Не нужно ничего знать о коде. Просто вставь текст ниже и отвечай на вопросы.

## Как пользоваться (3 шага)

### Шаг 1. Открой свою AI-среду
Любую из:
- **VS Code** + плагин Claude Code (или Cline)
- **Google Antigravity**
- **Claude Code** в терминале
- **ChatGPT Codex**

### Шаг 2. Вставь ЭТОТ промпт

```
Установи и запусти COMANDOS Starter Kit.
1) Склонируй репозиторий в домашнюю папку:
   - macOS / Linux:  git clone https://github.com/Comandosai/comandos-starter-kit ~/.comandos-kit
   - Windows (PowerShell): git clone https://github.com/Comandosai/comandos-starter-kit "$HOME\.comandos-kit"
   (если уже склонирован — обнови: git -C ~/.comandos-kit pull)
2) Прочитай файл bootstrap.md в этой папке и выполни его по шагам.
Общайся со мной на русском, простыми словами. Я не программист — делай всё за меня.
```

> На новом Mac первая команда `git` может открыть окно «установить инструменты
> разработчика» — это нормально, нажми «Установить» и подожди пару минут.

### Шаг 3. Отвечай на вопросы
Агент:
1. Проверит твой компьютер и доустановит что нужно (включая нужные MCP).
2. Спросит про базу знаний (где хранить знания о проектах). Нет — поможет создать.
3. Спросит, что хочешь сделать.
4. Развернёт проект и покажет, как запустить.

Готово. Дальше просто разговаривай с агентом — он знает структуру твоего проекта.

## Что можно создавать
Сайт, Telegram-бот, цифровой сотрудник (AI-агент), micro-SaaS, Telegram Mini App,
презентацию, видео, research-конспект — и многое другое. Если точного шаблона нет —
агент соберёт по правилам COMANDOS.

## Если что-то не получилось
Напиши агенту: «не сработало, объясни простыми словами что делать». Он поможет.
