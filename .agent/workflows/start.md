# /start — запустить COMANDOS Starter

Выполни `bootstrap.md` по шагам:
1. DOCTOR — `doctor/doctor.sh` (окружение, база знаний).
2. CONFIG+VAULT — создать/прочитать `~/.comandos/config.json`, привязать базу знаний.
3. ИНТЕРВЬЮ — новый/существующий проект + что делаем → выбрать рецепт.
4. РЕЦЕПТ — развернуть (template/repo/action), доустановить needs.
5. ФИНАЛИЗАЦИЯ — AGENTS.md + PROJECT-BRIEF.md, привязка к базе знаний, как запустить.

Общайся по-русски, просто. Делай всё за резидента.
